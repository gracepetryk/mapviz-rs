# mapviz-rs

A library for building beautiful 2D and 3D spatial visualizations on the web. It's built
with Rust, WebAssembly, WebGPU, and love :)

mapviz gives you the building blocks: coordinate systems, projections, cameras, tiled
basemaps, composable layers, and a streaming, time-aware data model. Point it at a GeoJSON
file for a static map, or pipe in a live feed of moving things and watch them animate
across a globe in real time, interpolated between updates.
